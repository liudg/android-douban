# android-douban

Material Design + Dagger2 + Rxjava + Retrofit + MVP 电影/图书APP

### 简介

一款遵循Material Design风格的电影、图书查询应用。已完成电影查询、登录注册、个人中心、收藏/取消收藏电影

- API来自豆瓣

- MVP模式

- RecyclerView展示电影/图书列表

- Dagger2依赖注入

- Retrofit（+RxJava）网络请求，数据缓存

- Rxjava实现事件总线

- 夜间模式
